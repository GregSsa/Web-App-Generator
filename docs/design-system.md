# Design system

Geist Sans porte l’interface et Geist Mono les métriques. Les surfaces utilisent les tokens shadcn en OKLCH avec un accent violet, thèmes clair/sombre et contraste élevé. Les cartes mobiles deviennent des grilles plus denses sur ordinateur. La navigation est inférieure sur mobile et latérale dès `lg`. Les zones principales mesurent au moins 44 px sur mobile ; les icônes proviennent de Lucide.

Chaque parcours prévoit chargement Skeleton, état vide, erreur compréhensible et confirmation Sonner. Les confirmations destructives doivent utiliser AlertDialog lors des prochaines opérations de suppression définitive.
